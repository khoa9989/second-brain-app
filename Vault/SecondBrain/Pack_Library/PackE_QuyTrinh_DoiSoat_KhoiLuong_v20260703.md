# Pack E – Quy trình Đối soát khối lượng thanh toán

**Mục tiêu:** Đối chiếu Nhật ký thi công ↔ Hợp đồng ↔ Shop Drawing trước khi thanh toán, đảm bảo khối lượng thanh toán đúng, đủ, hợp pháp.

**Phạm vi:** Các hạng mục: nền đường, móng, bê tông, cấu kiện, thiết bị.

**Nguyên tắc nạp:** Chỉ dùng căn cứ đã có trong `Legal_Master_Index.md`. Các điểm cần văn bản chính thức mới phải để trống hoặc ghi chú chờ cập nhật.

## 2. Biểu mẫu thực hiện

### Biểu 1: Đối soát khối lượng – Nhật ký ↔ Hợp đồng
| Hạng mục | ĐVT | SL theo Nhật ký | SL theo Hợp đồng | Chênh lệch | % Chênh | Ghi chú / Giải trình |
|---|---|---|---|---|---|---|
| Nền đường bê tông | m³ | | | | | Chờ bổ sung |
| Móng đá dăm | m³ | | | | | Chờ bổ sung |
| Bê tông cốt thép | m³ | | | | | Chờ bổ sung |
| Cốt thép | tấn | | | | | Chờ bổ sung |

**Quy tắc chênh lệch (dựa trên ND209/2026, ND210/2026 – chờ cập nhật):**
- ≤ 3%: Ghi nhận, thanh toán theo Nhật ký (có giải trình KST)
- 3-10%: Yêu cầu bổ sung biên bản làm việc, điều chỉnh hợp đồng phụ lục
- > 10%: Báo cáo Sở Xây dựng, xem xét xử lý theo quy định mới

### Biểu 2: Đối soát vật liệu đầu vào
| Vật liệu | Loại | Số lượng thực tế | Số lượng QC/Chứng nhận | Nguồn gốc | Trạng thái |
|---|---|---|---|---|---|
| Xi măng | PC40 | | | | Chờ bổ sung |
| Cốt thép | D16/D20 | | | | Chờ bổ sung |
| Đá 1x2 | m³ | | | | Chờ bổ sung |
| Cát xây dựng | m³ | | | | Chờ bổ sung |

### Biểu 3: Đối soát Shop Drawing vs Thiết kế
| Hạng mục | Vị trí trên Shop Drawing | Vị trí trên Thiết kế Kỹ thuật | Khác biệt | Xử lý |
|---|---|---|---|---|
| Cột C1 | | | | Chờ bổ sung |
| Dầm D2 | | | | Chờ bổ sung |
| Nền N3 | | | | Chờ bổ sung |

## 4. Mục bắt buộc chờ văn bản chính thức

| Mục | Văn bản đã có | Trạng thái xử lý |
|---|---|---|
| Quy trình ký phụ lục, điều chỉnh giá, xử lý chênh lệch khối lượng | ND210/2026/NĐ-CP | Cần rà soát điều khoản hợp đồng và cơ chế điều chỉnh |
| Thứ tự thanh toán, điều kiện thanh toán, trích khấu | ND209/2026/NĐ-CP | Cần rà soát điều khoản thanh toán/quyết toán |
| Tiêu chuẩn nghiệm thu, phương pháp kiểm tra, thời hạn bảo hành | ND207/2026/NĐ-CP | Cần rà soát điều khoản nghiệm thu và bảo hành |
| Chứng nhận vật liệu, quy trình kiểm tra vật liệu | TT32/2026/TT-BXD | Cần rà soát biểu mẫu và yêu cầu QC |

## 5. Checklist nhanh tại hiện trường

- [ ] Nhật ký thi công có đầy đủ ký hiệu, ngày giờ, người ghi
- [ ] Khối lượng Nhật ký khớp với khối lượng thực tế đo đạc
- [ ] Vật liệu có chứng nhận QC, tem kiểm tra, không quá hạn
- [ ] Shop Drawing đã được phê duyệt trước khi thi công
- [ ] Biên bản nghiệm thu hạng mục đã ký đủ chữ ký
- [ ] Có giải trình bằng văn bản cho mọi chênh lệch > 3%
- [ ] Thủ tục thanh toán đúng trình tự theo quy định mới
